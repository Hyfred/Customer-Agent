## 🚀 Running the Model

> **Overview:**  
> Run inference using either Bedrock API or local GPU deployment.

### Using Bedrock API

```bash
# Step 1: Update the API key in retrieve_wotool_infer_api.py

# Step 2: Execute all steps
bash run_all_step_api.sh
```

### Using Local GPU

```bash
# Run inference locally
bash run_all_step.sh
```
